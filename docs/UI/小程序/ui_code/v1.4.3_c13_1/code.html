甲、乙两车同时从 A 地出发，甲每小时行 60km，乙每小时行 50km，2 小时后甲比乙多行多少千米？
题号 学生答案 正确答案 错因依据
二(1) 20 千米 45 千米 速度差 × 时间
计算错误

错因诊断

  • 知识点：二元一次方程的应用-行程问题
  • 你的错误：把“速度差”当成了“速度和” 采用
  • 本质原因：没有理解“速度差 × 时间 = 路程差”的数量关系原理
  • 下次避免：先画示意图或写出数量关系，再列式计算
AI 提示

这类题的关键是先找出“速度差”，再乘上时间，
不要把“两车同时出发”理解错。
